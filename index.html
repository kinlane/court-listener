Court Opinion API

This is a working project to show was is possibile with the Free Law Project's Court Opinion API.

I'm work on a Swagger API definition for the API, server and client side tooling as well.

Currently this runs through an API proxy at court.listener.api.publicprivatesector.org so I can use CORS, I'm also using API keys instead of Basic Auth so I can provide analytics on usage, you can register over at API Evangelist API. The API is not listed in documentation, but the keys you are issued will work.
